# 📌 Project Overview

This project analyzes employee attrition using the **IBM HR Analytics Employee Attrition Dataset**.

The dashboard provides interactive visualizations that help HR professionals understand employee turnover, workforce composition, employee satisfaction, compensation trends and workforce planning.

---

# 🎯 Objective

To support HR decision-making through interactive dashboards by identifying patterns influencing employee attrition and workforce performance.

# 💡 Key Business Insights

- Employees working overtime show higher attrition.
- Sales and Research & Development departments experience comparatively higher employee turnover.
- Lower salary bands have higher attrition.
- Employee satisfaction positively impacts retention.
- Workforce demographics support succession planning and talent management.
